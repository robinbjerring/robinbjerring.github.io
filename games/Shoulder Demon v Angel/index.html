About Shoulder Demon v Angel

You have an hour to create and upload a game... But you will have to choose between good and evil actions along the way...

Game made for One Hour Game Jam. Game jam theme was Demons.

Had to make a game in one hour. Chose to make a choose your own adventure game. Wrote the story. Aslo wrote some code myself, but for this project I mainly refactored code from gamedev.tv course.
This is the first game jam were I worked alone.

Music "Mystical Theme" by Alexandr Zhelanov, https://soundcloud.com/alexandr-zhelanov
